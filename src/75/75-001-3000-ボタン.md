### ボタン

#### 日付選択

```py
import PySimpleGUI as sg

layout = [
    [sg.Text('Select a date')],
    [sg.Input(key='-Input-'), sg.CalendarButton('calender', target='-Input-')]
]

window = sg.Window('Calender', layout=layout)

while True:
    event, value = window.read()
    if event is None:
        break

window.close()
```
