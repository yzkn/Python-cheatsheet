### IPアドレス

#### IPアドレスを取得

```py
import ipaddress

# グローバルIPアドレス
import requests
global_ip = ipaddress.ip_address(
    requests.get('https://api.ipify.org').text
)
global_ip.is_global # True

# プライベートIPアドレス
import socket
private_ip = ipaddress.ip_address(
    socket.gethostbyname(socket.gethostname())
)
private_ip.is_private # True
```
