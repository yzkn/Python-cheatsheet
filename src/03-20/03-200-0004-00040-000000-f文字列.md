#### f 文字列

```py
one = 'first'
two = 2
three = '3rd'
mes = f'{one}: {two}{three}'
print(mes)
```

> first: 23rd

```py
strval=255
mes = f'value : {strval=:06d}'
print(mes)

# RAW文字列との組み合わせ
length=32
mes = fr'value :\t{strval=:0{length}o}'
print(mes)
```

> value : strval=000255
>
> # RAW 文字列との組み合わせ
>
> value :\tstrval=00000000000000000000000000000377
